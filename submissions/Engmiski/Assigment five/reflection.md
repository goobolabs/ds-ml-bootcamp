## 1. What I Implemented

I implemented a machine learning pipeline including data cleaning, preprocessing, feature engineering, and training three models: Logistic Regression, Random Forest, and Decision Tree.

## 2. Comparison of Models

Logistic Regression provided a stable baseline. Decision Tree showed overfitting behavior. Random Forest gave the best overall performance.

## 3. Random Forest Understanding

Random Forest is an ensemble method that combines multiple decision trees and uses majority voting to improve accuracy and reduce overfitting.

## 4. Third Algorithm

I used Decision Tree as the third algorithm because it is simple and easy to interpret, although it may overfit.

## 5. Metrics Discussion

Random Forest performed best across all metrics, while Logistic Regression was moderate and Decision Tree was weaker.

## 6. Final Findings

Random Forest is the best model for loan approval prediction because it balances accuracy and recall effectively.
